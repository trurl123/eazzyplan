Features

Ultra light project plan editor. It's a simple tool to create tree structure of tasks with automatic calculation of duration. All data stored in browser using HTML5 feature. Main feature is working with keyboard. It helps to create plans rapidly.

  • Autosave
  • Copy table to clipboard
  • Export/import
